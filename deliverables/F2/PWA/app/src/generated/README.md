# Generated code — do not hand-edit

Every file in this directory is written by `scripts/generate.ts` and overwritten on every `npm run generate`. Hand edits are lost.

## To change what lives here

Edit the relevant file under `spec/`, then run:

```bash
npm run generate
```

Commit the resulting diff alongside the spec change.

## Files

- `items.ts` — `sectionA` / `sections[]` constants from F2-Spec.md
- `schema.ts` — Zod schemas per section + inferred value types
