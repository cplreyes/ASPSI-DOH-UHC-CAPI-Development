## How to use this template

1. Copy this file to `Reports/{YYYY-MM-DD}-Weekly-Status.md` using the **Friday** date (send date).
2. Fill from `scrum/sprint-current.md` (current sprint) and `scrum/standups/*.md` (the week's daily entries).
3. Section 4 (Blocked / Needs ASPSI) is the most-read section — write it last so it's sharp.
4. Keep it to **1 page printed**. If it spills, cut Section 8 first, then Section 7.
5. Send Friday afternoon. ASPSI reads on Monday morning either way; Friday send signals discipline.
