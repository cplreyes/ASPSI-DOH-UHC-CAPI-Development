## Branch Strategy

```
main  ──── v2.0.0 ──────────────── ff-merge ──── v2.0.1
                                        ↑
                                       merge
                                        |
staging ─────── v2.0.0 ─── (Mon EOD) ─── v2.0.1-rc.1 ─── (Tue+) ─── (Fri)
                              ↑                ↑
                           merge from        merges from
                           feature           feature
                           branch            branch
                              |
                              |
f2-admin-portal-v2.0.1 ──── (Mon) ─── (per-fix commits) ─── (Fri)
   ↑
 branch from
 main @ v2.0.0
```

- **One feature branch** for all v2.0.1 work (`f2-admin-portal-v2.0.1`); commits flow Mon-Thu.
- **Daily merge to staging** (or every Tier-completion). Auto-deploys via `cf-pages-deploy.yml`.
- **Single ff-merge to main on Fri** at sprint close. v2.0.1 tag points at this merge commit.
- **Hotfix path** (if R2 surfaces CRITICAL during R3 cycle): branch off `main` not `staging`, merge to `main` → hotfix release `v2.0.0.1`, then cherry-pick to `staging` so R3's surface stays unmoved.

## Anti-Confusion Guardrails

| Risk | Mitigation |
|------|-----------|
| Tester files bug against wrong env | Two GH labels: `from-uat-round-2-2026-05` (exists) + `from-uat-round-3-2026-05-12` (new); R3 tester guide explicitly says which label to use |
| Tester mixes up env URLs | Pinned `#f2-pwa-uat` message + version banner in app (FX-015 verified working) |
| Same-tester credential confusion | Same usernames cross-env (`shan_admin`, `kidd_admin`), different passwords per env (already convention from R2 prep) |
| Bug fixed in v2.0.1 but still affecting R2 prod | Comment GH issue with v2.0.1 commit SHA; close on v2.0.1 release tag (Fri), not on staging merge |
| Schema migration drift | `runAllMigrations()` on staging only Mon-Thu; on prod only Fri at cutover |
| Demo data disturbance | Don't re-seed prod (DEMO-HCW-001..008) between R2 and v2.0.1 cutover |
| R2/R3 tester fatigue (Shan + Kidd both rounds) | Phase the day — R2 morning, R3 afternoon. Or split: Shan focuses R3, Kidd winds down R2. Carl's call with testers. |
